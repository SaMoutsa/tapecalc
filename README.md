# TapeCalc

A calculator web app with a running calculation history ("tape"), built with JavaScript.

## About

TapeCalc looks and behaves like a classic desktop calculator, but keeps a scrollable "tape" of every calculation you've made in the current session — so you can always check back on previous results.

## Features

- Standard arithmetic operations (+, −, ×, ÷) with correct operator precedence
- Parentheses support
- Percentage calculations
- Live expression preview
- Full keyboard support
- Calculation history ("tape") with a clear button
- No `eval()` — expressions are parsed with a custom tokenizer + recursive-descent parser
